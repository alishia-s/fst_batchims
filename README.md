# FST for mapping Korean Words to Romanized Counterparts for Liaison, Lateralization, Palatalization, and Nasalization Batchim Rules

This notebook works best in Google Colab.
1. Clone or download the repository
2. Open the jupyter notebook in Google Colab
3. Add the .txt files to the Colab by either a) mounting the your Google Drive and adding the files to your Drive or b) uploading them
4. Update the path that finds the .txt files in the notebook
5. Run all the cells
6. At the very bottom of the notebook, there is a place to input. You can enter any Korean word and see if the FST will output the Romanized version of the Korean word.

## Limitations
Can only use simple Korean words that follow the batchim rules:
1. Liaison
2. Lateralization
3. Palatalization
4. Nasalization
